Esegui questo comando dalla cartella cristian-pamz/ per generare i GIF:

  python3 generate_ascii.py tua_foto.jpg

Richiede: pip install Pillow

I file generati saranno:
  assets/ascii-dark.gif   (sfondo scuro, testo chiaro)
  assets/ascii-light.gif  (sfondo chiaro, testo scuro)

Carica questi file su GitHub insieme al README.md
